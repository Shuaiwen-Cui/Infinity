# OPENSEES

!!! info
    OpenSees 是一个用于模拟结构和岩土系统在地震作用下的性能的应用程序的软件框架。它是由加州大学伯克利分校的Gregory L. Fenves教授领导的太平洋地震工程研究中心（PEER）开发的开源软件。
