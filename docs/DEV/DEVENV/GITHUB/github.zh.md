# GITHUB

## 什么是 GitHub 以及为什么使用它？

GitHub 是一个用于版本控制和协作的基于 Web 的平台。它是微软的子公司，微软于 2018 年以 75 亿美元收购了该公司。它提供了 Git 的分布式版本控制和源代码管理（SCM）功能，以及它自己的功能。
