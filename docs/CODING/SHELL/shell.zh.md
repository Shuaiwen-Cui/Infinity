# 终端解释器 SHELL

## 什么是SHELL

Shell是一个从键盘接收命令并将其传递给操作系统执行的程序。在旧时代，它是类Unix系统（如Linux）上唯一的用户界面。现在，我们除了命令行界面（CLI）如shell外，还有图形用户界面（GUI）。
