# INFLUXDB

## 什么是 InfluxDB？

InfluxDB 是由 InfluxData 开发的开源时间序列数据库。它是用 Go 语言编写的，专为快速、高可用性的时间序列数据存储和检索而优化，适用于运维监控、应用程序指标、物联网传感器数据和实时分析等领域。
