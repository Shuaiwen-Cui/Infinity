# POSTGRESQL

## 什么是 PostgreSQL?

PostgreSQL 是一个开源的面向对象的关系型数据库系统，拥有超过 15 年的活跃开发历史。它拥有经过验证的架构和良好的可靠性、数据完整性和正确性的声誉。
