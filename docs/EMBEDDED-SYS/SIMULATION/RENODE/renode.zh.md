# RENODE

## 简介

[Renode](https://renode.io/) 是一个用于嵌入式设备、系统和网络的多节点仿真的开源框架。它可以让你创建一个具有无限可能的可配置节点的虚拟设备网络。你可以交互式地调试你的多节点仿真，也可以使用内置的框架创建自动化测试和回归套件。

## 官方网站简介

### Renode 是什么？

**Renode** 是一个开源软件开发框架，由 Antmicro 提供商业支持，它可以让你可靠、可扩展、有效地开发、调试和测试**多节点设备系统**。

Renode 的优势：

- 执行的完全确定性，共享虚拟时间
- 透明且强大的调试、跟踪、分析，即使在多节点设置中也是如此
- 与你的日常工具、插件的轻松集成
- 丰富的模型抽象，附加功能“免费”+ - 模块化平台描述格式
- 自动化测试和 CI 集成，其他协作功能

### 主要特性

- 可扩展的 IoT 仿真框架。
- 支持现代、快速的开发工作流程。
- 大大提高了测试能力。
- 让你开发复杂的多节点场景。
- 为持续集成而建。
- 易于使用、开放和可扩展。
- 实现安全硬化的设备和系统。
